# Edentic Logo Pack

Contents:
- `icon/`: abstract Edentic E mark
- `wordmark/`: Edentic text-only logo
- `combined/`: icon + wordmark lockup
- `social/`: ready-to-use social and Open Graph images
- `source-originals/`: the three supplied source images

Formats:
- Transparent PNG
- White-background PNG
- JPG
- SVG wrappers with embedded high-resolution transparent artwork
- Multi-resolution favicon ICO

Notes:
- Raster assets are centered with safe padding to prevent clipping at small sizes.
- JPG files use a white background because JPG does not support transparency.
- The SVG files preserve the supplied artwork exactly by embedding the cleaned high-resolution PNG.
